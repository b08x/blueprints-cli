Installation & Usage Guide

1. Clone Repository

Get the source code on your local machine.

git clone https://github.com/b08x/blueprints-cli.git
cd blueprints-cli

2. Install Dependencies

Install the required Ruby gems using Bundler.

bundle install

3. Set Up Database

Create and migrate the PostgreSQL database.

rake db:create
rake db:migrate

4. Configure Environment

Set your API key and database URL. Create a .env file or export variables.

GEMINI_API_KEY="your-gemini-api-key"
DATABASE_URL="postgresql://user:pass@host:5432/blueprints_development"

5. Run a Command

You're ready to go! Try submitting your first blueprint.

bin/blueprintsCLI blueprint submit "puts 'Hello, Blueprints!'"
